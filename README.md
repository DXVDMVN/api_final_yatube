# api_final
api final
Этот проект представляет собой API для социальной сети YaTube — платформы для публикации блогов. API обеспечивает взаимодействие между клиентами и сервером, обрабатывая стандартные запросы для следующих действий:

создание, редактирование, удаление и просмотр публикаций;

управление комментариями к публикациям;

получение информации о сообществах;

подписка на других пользователей.

API доступен только авторизованным пользователям. Каждый пользователь может изменять и удалять только свой контент — чужой доступен только для чтения. Анонимные пользователи могут лишь просматривать общедоступные данные, за исключением информации из подписок.
